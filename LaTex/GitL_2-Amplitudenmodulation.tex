\section{Vorbereitung}
\subsection{Was ist der Unterschied zwischen modulierendem und moduliertem Signal?}
Ein Moduliertes Signal ist das Ergebnis einer Modulation \cite{}.
Dabei wird durch ein höherfrequentes Trägersignal das Modulierenden Signal dahingehen verändert, damit es übertragen werden kann. \\
Beim Modulierenden Signal handelt es sich um das Nachrichten oder Nutzsignal \cite{}. 
Dieses enthält die die Information und soll übertragen werden.

\subsection{Wie lautet für ein sinusförmiges Trägersignal $x_0(t) = U_0 sin(\omega_0t + \phi 0)$ der Zusammenhang zwischen der Amplitude $U_0$ und dessen Effektivwert?}
Für sinusförmige Signale, somit auch für Trägersignale gilt \cite{Leifiphysik_wechselstrom}:
\begin{equation}
    U_{eff}=(\frac{U_0}{\sqrt{2}})
    \label{Gl:Effektivwert_sin}
\end{equation}
Dabei gibt der Effektivwert den Gleichspannungswert, der an einem ohmschen Widerstand die gleiche Leistung erzeugen würde.\\
Da bei der Frequenzmodulation sowie Phasenmodulation die Amplitude konstant ist, kann die Gleichung \ref{Gl:Effektivwert_sin} auch dabei verwendet werden.\\
Bei der Amplitudenmmodulation kann der Effektivwert nicht so leicht berechnet werden, da hier die Amplitude schwankend ist. 
Für diesen Fall kann folgende Formel zur berechnung des Effektivwertes verwendente werden \cite{Ampl_eff}.
\begin{equation}
    U_{eff}=\frac{U_0}{\sqrt{2}} \cdot \sqrt{1+\frac{m^2}{2}} 
\end{equation}

\subsection{Berechnung der Resonanzfrequenz des Parallelschwingkreises}

Der gegebene Parallelschwingkreis besteht, zu sehen im Schaltplan Abbildung \ref{Ah:Schaltplan_Platine}, aus einer Spule $L_1$ und drei parallel geschalteten Kondensatoren $C_{16}$, $C_{17}$ und $C_{18}$.


\begin{align*}
    L_1    &= 22\,\mu\text{H} = 22 \cdot 10^{-6}\,\text{H} \\
    C_{16} &= 4\text{n}7 = 4,7\,\text{nF} = 4,7 \cdot 10^{-9}\,\text{F} \\
    C_{17} &= 220\,\text{pF} = 220 \cdot 10^{-12}\,\text{F} = 0,22 \cdot 10^{-9}\,\text{F} \\
    C_{18} &= 0\,\text{pF} \quad \text{(optionaler Bestückungsplatz)}
\end{align*}
 
\subsubsection{Berechnung der Gesamtkapazität}
Bei einer Parallelschaltung von Kondensatoren addieren sich die Einzelkapazitäten zur Gesamtkapazität $C_{\text{ges}}$ auf \cite{hagmannGrundlagenElektrotechnikBewaehrte2020}:
\begin{align*}
    C_{\text{ges}} &= C_{16} + C_{17} + C_{18} \\
    C_{\text{ges}} &= 4,7\,\text{nF} + 0,22\,\text{nF} + 0\,\text{nF} = 4,92\,\text{nF} \\
\end{align*}

\subsubsection{Berechnung der Resonanzfrequenz}
Die Resonanzfrequenz $f_r$ des Parallelschwingkreises wird mithilfe der Thomson-Schwingungsformel ermittelt \cite{hagmannGrundlagenElektrotechnikBewaehrte2020}:
\begin{equation*}
    f_r = \frac{1}{2\pi \cdot \sqrt{L_1 \cdot C_{\text{ges}}}}
\end{equation*}


\begin{equation*}
f_r = \frac{1}{2\pi \cdot \sqrt{22 \cdot 10^{-6}\,\text{H} \cdot 4,92 \cdot 10^{-9}\,\text{F}}} \approx 483.756\,\text{Hz} \approx 483,8\,\text{kHz}
\end{equation*}


\subsubsection{Resonanzfrequenz}
Die theoretische Resonanzfrequenz des Parallelschwingkreises beträgt ca. \textbf{483,8\,kHz}.

\subsection{Programmcode}

\begin{lstlisting}[style=pythonstyle,    
    caption={Pyhtoncode zum Erzeugen von Bodeplotts},
    label={lst:Bodeplotts_erzeugung}]
    import numpy as np
    import matplotlib.pyplot as plt
    from scipy import signal

    # Parameter
    C = 4.92e-9  # 4.92 nF
    L = 22e-6    # 22 uH

    # Resonanzfrequenz berechnen
    f0 = 1 / (2 * np.pi * np.sqrt(L * C))
    omega0 = 2 * np.pi * f0

    print("=" * 60)
    print("PARALLELSCHWINGKREIS - UEBERTRAGUNGSFUNKTION")
    print("=" * 60)
    print(f"\nGegebene Werte:")
    print(f"  C = {C*1e9:.2f} nF")
    print(f"  L = {L*1e6:.2f} uH")
    print(f"\nBerechnete Werte:")
    print(f"  Resonanzfrequenz f0 = {f0/1e6:.4f} MHz")
    print(f"  Kreisfrequenz omega0 = {omega0:.2e} rad/s")

    Q = 10

    print(f"  Guetefaktor Q = {Q:.2f}")

    # Uebertragungsfunktion:
    # H(s) = omega0^2 / (s^2 + (omega0/Q)*s + omega0^2)

    num = [omega0**2]
    den = [1, omega0/Q, omega0**2]

    print("\n" + "=" * 60)
    print("UEBERTRAGUNGSFUNKTION")
    print("=" * 60)
    print(f"\nH(s) = omega0^2")
    print(f"       -------------------------")
    print(f"       s^2 + (omega0/Q)*s + omega0^2")

    frequency_ratio = np.logspace(-1, 0.8, 1000)
    f = frequency_ratio * f0
    w = 2 * np.pi * f

    w_response, H_jw = signal.freqs(num, den, w)

    magnitude = np.abs(H_jw)
    phase = np.angle(H_jw) * 180 / np.pi
    magnitude_norm = magnitude

    fig = plt.figure(figsize=(14, 10))
    gs = fig.add_gridspec(2, 1, height_ratios=[2, 2], hspace=0.3)

    ax1 = fig.add_subplot(gs[0])
    ax1.semilogx(frequency_ratio, magnitude_norm, 'b-', linewidth=2.5)

    ax1.set_xlim([0.1, 6])
    ax1.grid(True, which='both', alpha=0.3)
    ax1.set_ylabel('|H(jw)|', fontsize=12, fontweight='bold')
    ax1.set_title(
        f'Betragsfrequenzgang (C={C*1e9:.2f}nF, L={L*1e6:.2f}uH)',
        fontsize=14,
        fontweight='bold'
    )

    ax3 = fig.add_subplot(gs[1])
    ax3.semilogx(frequency_ratio, phase, 'b-', linewidth=2.5)

    ax3.set_xlim([0.1, 6])
    ax3.set_ylim([-180, 0])
    ax3.grid(True, which='both', alpha=0.3)
    ax3.set_xlabel('f/f0', fontsize=12, fontweight='bold')
    ax3.set_ylabel('phi(w) [deg]', fontsize=12, fontweight='bold')
    ax3.set_title(
        'Phasengang (logarithmische Darstellung)',
        fontsize=12,
        fontweight='bold'
    )

    plt.savefig(
        'parallelschwingkreis_frequenzgang.png',
        dpi=150,
        bbox_inches='tight'
    )

    print("\nPlot gespeichert")
    plt.show()

    print("\n" + "=" * 60)
    print("CHARAKTERISTISCHE WERTE")
    print("=" * 60)

    idx_resonance = np.argmin(np.abs(frequency_ratio - 1.0))

    print(f"\nBei Resonanz (f/f0 = 1.0):")
    print(f"  y/y0 = {magnitude_norm[idx_resonance]:.4f}")
    print(f"  Phase = {phase[idx_resonance]:.2f} deg")
\end{lstlisting}

with open("tabelle.txt", "r", encoding="utf-8") as file:
